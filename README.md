# GWAS-Prediction-IBD-T2D: Genetic Risk Prediction using Machine Learning

This repository contains the complete codebase, data cleaning pipelines, and final reports for a project that utilizes Machine Learning (ML) and Genome-Wide Association Study (GWAS) data to predict genetic risk for two major diseases.
